OrderedCollection

OrderedCollection

new OrderedCollection(iterableOrGenerator, keySelector)

OrderedCollection - Represents an ordered collection of iterable values providing additional methods to order an already ordered collection a second time keeping the order of non-equal elements
Parameters:
Name Type Description
iterableOrGenerator Iterable | GeneratorFunction A iterable to create a collection of, e.g. an array or a generator function
keySelector function | String A function which maps to a property or value of the objects to be compared or the property selector as a string

ThenBy(keySelector) → {OrderedCollection}

Performs a subsequent ordering of the elements in a sequence in ascending order according to a key. The default comparator is used to compare values.
Parameters:
Name Type Description
keySelector function | String A function which maps to a property or value of the objects to be compared or the property selector as a string

Type
OrderedCollection
Example
const pets = [
   {
     Name: 'Barley',
     Age: 8,
   },
   {
     Name: 'Boots',
     Age: 1,
   },
   {
     Name: 'Whiskers',
     Age: 1,
   },
   {
     Name: 'Fluffy',
     Age: 2,
   },
   {
     Name: 'Donald',
     Age: 4,
   },
   {
     Name: 'Snickers',
     Age: 13,
   }
 ]

 pets.OrderBy(x => x.Name.length).ThenBy(x => x.Age).Select(x => x.Name).ToArray()
 // -> ["Boots", "Fluffy", "Donald", "Barley", "Whiskers", "Snickers"]
